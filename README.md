# Python ATM Simulator

A simple ATM simulation app built with Python.

This project was created to practice basic programming logic, user input, conditions, loops, functions and simple banking operations.

## Features

- PIN check
- Balance check
- Deposit money
- Withdraw money
- PIN change
- Basic transaction flow
- Simple terminal interface

## Technologies Used

- Python

## What I Practiced

- Variables
- User input
- If / elif / else logic
- Loops
- Functions
- Basic error handling
